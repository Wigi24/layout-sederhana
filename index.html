Pengertian HTML, CSS, PHP, Javascript

HTML

HTML (Hypertext Markup Language) adalah bahasa markah standar untuk dokumen yang dirancang untuk ditampilkan diperamban internet.

CSS

CSS adalah bahasa (Casacading Style Sheet) dan biasanya digunakan untuk mengatur tampilan eleman yang tertulis dalam bahasa markup, seperti HTML.

PHP

PHP (Hypertext Preprocessor) atau hanya PHP saja, adalah bahasa skrip dengan fungsi umum yang terutama digunakan untuk pengembangan web.

JavaScript

JavaScript adalah bahasa pemrograman tingkat tinggi dan dinamis.
